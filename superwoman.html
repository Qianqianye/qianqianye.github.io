Superwoman

2017

I am really excited to be part of 'Diversity with Code+Art' project created by Chelly Jin and powered by P5.js.

This sketch 'Superwoman' is about empowerment of women in STEM, sending a message that' it is never just a dress'. My work will be featured on the p5js.org homepage from May 3rd to May 17th, so please check it out if you like using software to make strange/cute things. Cute is good.

You can also hear me rambling about architecture / urban / code / VR / data in the interview session.
